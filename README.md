# Statistical-Modelling-of-Prostate-Cancer-Biomarkers
This project models Log Prostate Specific Antigen (lpsa) using clinical data from 97 patients. By comparing OLS, Robust Regression, Lasso, and GAM, the study identifies Lasso Regression as the optimal model (Test RMSE: 0.7037), highlighting tumor volume as a critical predictor for risk stratification.
